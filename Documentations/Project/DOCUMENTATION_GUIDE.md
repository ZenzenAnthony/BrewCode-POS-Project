# Documentation Guide

This document serves as a guide for finding and using documentation in the BrewCode POS Project.

> **Note:** For a more detailed documentation structure overview, see `Documentations/DOCUMENTATION_STRUCTURE.md`

## Documentation Structure

1. **Database Documentation** - Located in the `Database` folder:
   - `COMPLETE_MENU_INVENTORY_RELATIONSHIPS.md` - Master document listing all menu-inventory relationships
   - `DATABASE_UPDATE_SUMMARY.md` - Summary of database structure updates
   - `MENU_INVENTORY_RELATIONSHIP.md` - Explanation of menu-inventory relationship design
   - `MENU_DRINK_UPDATE.md` - Documentation for the drink menu item updates
   - `MENU_INVENTORY_ENHANCEMENT_SUMMARY.md` - Summary of menu inventory relationship enhancements
   - `FINAL_IMPLEMENTATION_SUMMARY.md` - Final summary of all implementations

2. **CSV Data Documentation** - Located in the `CSV` folder:
   - `README.md` - General information about the CSV files and usage
   - `MENU_ITEM_INGREDIENTS_DOCS.md` - Specific documentation for the menu-item-ingredients.csv file
   - `SAMPLE_DATA_REFERENCE.md` - Reference document showing sample data relationships

## Consolidated Structure

The documentation has been organized to avoid duplication:
- Database-related concepts and design documentation are in the Database folder
- Data format and CSV usage documentation are in the CSV folder

1. For understanding database structure and relationships, start with the documents in the Database folder
2. For working with the CSV files directly, refer to the documentation in the CSV folder
3. For implementation details of the menu-inventory relationships, see the COMPLETE_MENU_INVENTORY_RELATIONSHIPS.md file
